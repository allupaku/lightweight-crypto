Performance on AVR

This page lists the performance figures for AVR platforms, in particular the ATmega2560 running at 16MHz. Figures for other AVR chips should be fairly similar. Performance figures for 32-bit platforms can be found here.

These figures are very preliminary, as most of the figures are based on the 32-bit plain C versions of the algorithms. A lot of work is needed to make the algorithms suitable for the 8-bit AVR platform.

An asterisk * indicates algorithms that have since been accelerated with AVR assembly code or alternate C implementations that are tuned for 8-bit platforms. The remaining algorithms are still in plain C and designed for 32-bit.
